Instalación en Windows:
1) Descargar python v3.5 de: https://www.python.org/downloads/release/python-350/
2) Alternativa 1:
  a) Carpeta de PLY 3.8 ya incluida.
  b) Ejecutar setup.py que se encuentra en Cheese/ply-3.8
3) Alternativa 2:
  a) Ejecutar desde la línea de comandos: python -m pip install ply
  b) PLY se debe de descargar e instalar automáticamente.

Compilación:
1) Abrir cmd.
2) Navegar al directorio donde se encuentra la carpeta Cheese.
3) Una vez dentro ejecutar: py parser.py <nombre_del_archivo.txt>
4) Se desplegará en pantalla si se compiló correctamente o hay algún error.

Ejecución:
1) Una vez ya compilado el código.
2) Abrir cmd y navegar al directorio de la carpeta Cheese.
3) Ejecutar: py virtualMachine.py cheeseCode.rekt
   cheeseCode.rekt es el nombre del código intermedio que se genera.
4) Observar lo que se despliega en consola o pantalla del graficador.
